# PSPO-I 参考書ガイド

このディレクトリは、PSPO I を「受験対策本」ではなく「価値判断の語彙を手に入れる参考書」として使うための教材である。
目的は合格証を取ることではなく、顧客価値、優先順位、Backlog、Goal、ステークホルダー調整を自分の案件文脈で説明できるようになることにある。

## この教材の前提

- 今回の資格戦略では、PSPO I の受験は見送る
- 必要な語彙だけを学習で回収する
- 受験の主目的は置かない
- CAPM 前に「何をやるべきか」を語るための土台として使う

## この教材の使い方

- まず「用語の意味」を覚える前に、「その概念がないと何が壊れるか」を考える
- 各章のケースを、自分の案件に置き換えて読む
- 「受験するかどうか」ではなく、「この語彙を現場で使えるか」を確認基準にする
- 各章の最後にある一文を、実務でそのまま言えるレベルまで短くする

## 到達目標

- 顧客価値を、単なる要望の強さと切り分けて説明できる
- Product Backlog とタスクリストの違いを説明できる
- Product Goal と Sprint Goal の違いを説明できる
- ステークホルダー調整を「誰の声が強いか」ではなく「何に効くか」で整理できる
- 「なぜ PSPO I は今回は受験見送りで十分か」を説明できる

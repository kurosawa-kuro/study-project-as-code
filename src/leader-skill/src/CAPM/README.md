## 合格のための最重要ポイント

1. **EVM公式を計算できるようにする**（CV・SV・CPI・SPI・EAC・ETC・TCPI）
2. **BAドメインを後回しにしない**（要求収集・RTM・受入基準は27%の中核）
3. **アジャイルを後回しにしない**（Scrum・Kanban・適応型判断は20%の中核）
4. **PMIの思考パターンを覚える**（先手・プロセス通り・ステークホルダー優先）
5. **模擬問題で70%以上を安定させる**

## この教材の到達点

- EVM公式を見ずに計算できる
- CPMでクリティカルパスとフロートを求められる
- リスク対応戦略8種（脅威4＋機会4）をシナリオで即答できる
- Scrumの役割・アーティファクト・セレモニーを混同なく説明できる
- BAの役割・要求収集技法・RTM・受入基準をシナリオで判断できる
- 演習問題で70%以上を取れる

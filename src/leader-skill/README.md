# leader-skill

リモート技術リードとしての信用補強に必要な資格戦略を整理する子プロジェクト。
アプリケーション実装が主目的ではなく、資格方針・ロードマップ・運用ルールを明文化し、判断のぶれを減らすことを目的とする。

旧 `study-leader-skill` リポジトリを Project as Code（親リポジトリ）へ履歴ごと吸収したもの。ハーネス・秘密情報管理・コミット規約は親リポジトリに従う。

## 現在の位置づけ

- 主対象: リーダースキル資格の仕様整理と学習教材
- 主眼: PM 専業化ではなく、技術主軸を保ったまま外部証明を補強すること
- 現在の結論: `PMLE -> PSPO I 学習のみ -> CAPM 取得 -> Databricks 基礎資格`

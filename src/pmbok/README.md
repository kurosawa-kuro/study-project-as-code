# pmbok

PMBOK を中心とした PM 知識体系を整理する子プロジェクト。Project as Code（親リポジトリ）が管理する。

資格試験対策そのものではなく、PMBOK の概念・プロセス・用語を実務（PM / リーダー業務、PaC 運用）へ接続できる形で正本化することを目的とする。

## 位置づけ（暫定）

- 主対象: PMBOK 第7版を軸とした PM 知識体系の整理（第6版のプロセス群・知識エリアは必要な範囲で補助的に参照）
- 主眼: 「試験で問われる形」ではなく「実務とAI協働で使える形」への再構成
- 隣接プロジェクトとの役割分担:

| プロジェクト | 役割 |
|---|---|
| `src/pmbok/` | PM 知識体系の正本整理（このプロジェクト） |
| `src/leader-skill/` | 資格戦略と試験対策教材（CAPM / PSPO I）。試験特化の知識はこちら |
| 親 `docs/specs/pm-leader-skill-model.md` | PM スキルモデルの PaC 用語への正規化 |

CAPM 学習（`src/leader-skill/src/CAPM/`）と重複する概念は、体系側の正本をここに置き、試験対策側から参照する方向で育てる。
